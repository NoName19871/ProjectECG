Назначения элементов: 
-Кнопка Enter data открывает DataForm для дальнейшего ввода значений кардиограммы 
-Кнопка Analyse позволяет поставить диагноз в соответствии с введенными значениями 
-Кнопка Help выводит справку по правилам работы с MyForm 
-Кнопка Save to file сохраняет введенные данные в csv файл 
-Выпадающие списки позволяют выбрать необходимые для построения графика отведения 
-Кнопка Create ECG graphic строит 3 графика по отведениям, выбранным из выпадающих списков 
-В поле Analysis result выводится поставленный диагноз, а также частота сердечных сокращений пользователя
